## Installation command
This command works for Arch Linux, macOS, and other Unix-like operating systems.

For Windows or alternative installation methods, see:  
https://forge.rust-lang.org/infra/other-installation-methods.html

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## 3. Verifying the installation
To verify that you install Rust correctly, run this commands in your bash:

```bash
rustc --version
cargo --version
```

This must give you a message similar than the next:
```bash
rustc 1.92.0 (ded5c06cf 2025-12-08)
cargo 1.92.0 (344c4567c 2025-10-21)
```
